SpoonRocket Data Analysis

I assisted SpoonRocket with data analysis as a finance intern. My first project concerned forecasting, where I used Excel to measure the growth of orders week over week to properly gauge future supply. I then created a model for tracking and analyzing the number of active drivers, which required me to learn and use a series of window functions in PostgreSQL. My paramount project dealt with measuring historical sold out times using a Python script to parse CSV files for relevant data.
